# 🎮 Steam Games Market Analysis Dashboard

## 📋 Overview
Analysis of 122,000+ Steam games using Python and Power BI, covering market, quality, and popularity metrics.

## 🛠️ Tools Used
- **Python** (Pandas) — data cleaning and transformation
- **Power BI** — dashboard and visualization

## 📊 Dashboard Metrics
- Total games, average price, recommendations
- Games released per year
- Top 5 genres, developers, publishers
- Top 5 most expensive games
- Top 5 games by Peak CCU
- Top 5 most recommended games
- Top 5 games by Metacritic score

## 📁 Dataset
Source: [Steam Games Dataset - Kaggle](https://www.kaggle.com/)

## 🚀 How to Run
1. Download the dataset from Kaggle
2. Run `steam_analysis.py` to generate `games_clean.csv`
3. Open `Steam_Games_Dashboard.pbix` in Power BI
